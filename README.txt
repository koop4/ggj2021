# Getting Started
---------------
1 Clone your repo
	git clone git@github.com:koop4/ggj2021.git
2.Download Unity and Unity GitHub Plugin
	https://unity3d.com/get-unity/download
	https://unity.github.com/
3.Open UnityHub add the repository and open the project (make sure you running v2019.4.18f1) 
4.Add your repo to Unity Hub and open the project in Unity
5.install Unity GitHub plugin
